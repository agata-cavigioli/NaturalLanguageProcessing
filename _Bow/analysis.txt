To represent the similarity of the documents I've chosen to use an upper triangular matrix, to save computation. The diagonal elements (i.e., where the row index equals the column index) are set to -1, as well as the lower triangular parts, that would have been symmetric. The off-diagonal elements contain similarity values ranging from 0 to 1. These values represent how similar each pair of documents is to each other based on the cosine similarity measure. Higher values indicate greater similarity, while values close to 0 indicate dissimilarity.
